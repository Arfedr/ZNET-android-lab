\# ZNET Android Lab



Лабораторный репозиторий для сравнения и переноса TUN/VPN пайплайна из v2rayNG в AndroidLibXrayLite-Z-NET.



\## Репозитории

\- deps/v2rayNG — референс (как должно работать)

\- deps/AndroidLibXrayLite-Z-NET — цель (куда переносим)



\## Как открыть в Android Studio

Открывать в двух окнах:

1\) deps/v2rayNG

2\) deps/AndroidLibXrayLite-Z-NET



\## Цель

создать свое приложение по типу happ / v2raytun, чтобы было подключение vless и всех остальных из v2rayng и могли быть reality ну и тоже все остальное под новое ядро xray-core из AndroidLibXrayLite-Z-NET.
